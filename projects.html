Submarine Game

This was a project for a web programming course in college that I had a lot of fun with. The basic premise was to create a Space Invaders style game in which you shoot projectiles from a player controlled ship and try to hit enemies falling from the top of the screen. In my version however, the player controlls a submarine, and shoots torpedos at sharks. What I enjoyed most about this project was adding lots of randomness to the game mechanics. For example, the sharks spawn at random points along the top of the screen, the movement of the sharks is randomized, and the initial speed and acceleration of each torpedo that the player fires is also random. I learned a lot about the basic structure of a game through this project, as well as the capabilities of JavaScript. The game is playable your browser, so if you'd like to check it out, it can be played here!

Eventually, I may come back to this project to improve it or use it as a touchstone for creating other basic games that could be played in a browser.
